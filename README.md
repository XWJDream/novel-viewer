# TXT 文件阅读器

一个轻量级的本地 TXT 文件 Web 阅读器，专为批量浏览文件夹中的文本文档设计。零依赖，开箱即用。

## 快速开始

```bash
# 默认打开当前目录
node server.js

# 指定任意文件夹
node server.js "E:\my-folder"
```

打开浏览器访问 http://localhost:3000

## 功能

### 文件浏览
- 左侧边栏展示当前文件夹的所有 txt 文件
- 支持浏览子文件夹、返回上级目录
- 顶部显示当前文件夹路径，点击"更换"可切换到任意文件夹

### 智能分类

自动识别文件命名模式并分组：

| 命名模式 | 示例 | 分组方式 |
|----------|------|----------|
| 章节 | `第1章.txt`、`8-正文-第3章.txt` | 章节文件 → "正文章节"，其余 → "设定与资料" |
| 数字前缀 | `01-会议.txt`、`02-会议.txt` | 按前缀数字分组（同前缀 ≥2 个文件时生效） |
| 日期前缀 | `2024-01-01-日记.txt` | 按年月分组 |
| 括号前缀 | `【工作】报告.txt` | 按括号内容分组 |
| 通用前缀 | `project-A.txt`、`project-B.txt` | 共享 `-` 或 `_` 前缀的文件自动归组 |

无法识别的文件平铺显示在"文件"分类下。

### 阅读体验
- 中文排版优化：1.9 倍行高、两端对齐、段首缩进
- 章节底部"上一章 / 下一章"导航
- 字体大小调节（A+ / A-）
- 深色 / 浅色主题切换
- 底部滚动进度条

### 搜索
- 文件名实时过滤
- 全文内容搜索，关键词高亮，点击结果直接跳转
- 快捷键：`Ctrl + /` 聚焦搜索框，`Esc` 关闭

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | 返回页面 |
| GET | `/api/files` | 当前文件夹的文件列表（含分类、子目录） |
| GET | `/api/file?name=xxx` | 读取指定 txt 文件内容 |
| POST | `/api/set-folder` | 切换文件夹，body: `{"folder":"path"}` |
| GET | `/img/xxx` | 返回当前文件夹中的图片 |

## 环境要求

- Node.js（已在 v24 测试，理论上 v14+ 均可）
- 无需 npm install，无第三方依赖
